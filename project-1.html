URL Shortener

A light-weight web application built using Flask to shorten the URLs to 10 characters.

GitHub Link

Project Overview

No one likes to type in or work with long URLs. URL shortener basically provides its user with a short URL which when requested will redirect its user to the original website. This is done by mapping the long URL string provided by user to a smaller string of characters.

In our case, the Flask web application provides an interface for its user to provide the URL that they want to shorten. When they click submit, the tool will generate a random string and map it with original Database. The mapping is stored in SQLAlchemy database. When user tries to open the shortened URL, the tool get the original URL from the mapping and redirect its user to the original web page.

The tool does all the standard checks like checking if the newly generated random string is already mapped. If it is, the tool will generate a new string again. It will repeat the process until a new unique string is found. The tool also uses the standard WTForms and CSRF protection is also implemented.
